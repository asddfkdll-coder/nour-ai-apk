# منصة نور (Nour)

منصة متقدمة للتفاعل مع الشخصيات الرقمية، مبنية باستخدام أحدث التقنيات لضمان تجربة مستخدم سلسة وآمنة.

## 🚀 التقنيات المستخدمة

- **الواجهة الأمامية:** React 19 + Tailwind CSS 4 + Vite
- **الخلفية:** Express + tRPC
- **قاعدة البيانات:** Drizzle ORM + MySQL
- **الأمان:** Rate Limiting + Security Headers + Input Validation

## 🏗️ هيكلية المشروع

- `client/`: كود الواجهة الأمامية.
- `server/`: كود الخادم والمنطق البرمجي.
- `drizzle/`: تعريفات قاعدة البيانات.
- `shared/`: الأنواع والثوابت المشتركة.

## 🛠️ البدء في التشغيل

1. تثبيت التبعيات:
   ```bash
   pnpm install
   ```

2. تشغيل المشروع في بيئة التطوير:
   ```bash
   pnpm dev
   ```

3. بناء المشروع للإنتاج:
   ```bash
   pnpm build
   ```

## 🔐 الأمان والخصوصية

تم تصميم المنصة مع التركيز على خصوصية المستخدم وأمان البيانات، حيث يتم تشفير كافة المحادثات وتطبيق سياسات صارمة لحماية البيانات.
